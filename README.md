# 🎓 Sistema de Cadastro de Alunos

Projeto desenvolvido para a disciplina de Aplicações Front-End utilizando Angular e JSON Server.

O sistema permite realizar operações completas de CRUD (Create, Read, Update e Delete) sobre alunos cadastrados, simulando uma API REST através do JSON Server.

---

# 📋 Objetivo do Projeto

Desenvolver uma aplicação Front-End capaz de:

- Cadastrar alunos
- Listar alunos cadastrados
- Editar informações de alunos
- Excluir alunos
- Consumir uma API REST
- Persistir dados em arquivo JSON
- Utilizar boas práticas de organização com componentes, services e models

---

## Tecnologias Utilizadas

- Angular

## Funcionalidades

- Cadastro de alunos
